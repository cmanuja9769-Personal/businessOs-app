## ğŸ“¡ Backend API Endpoints (Already Implemented)

All endpoints require JWT authentication via `Authorization: Bearer <token>` header.

### Authentication
```
POST /api/v1/auth/login
Headers: { Content-Type: application/json }
Body: { email, password }
Response: { success, data: { token, user } }
```

### E-Way Bill Endpoints

1. **Generate E-Way Bill**
   ```
   POST /api/v1/e-waybill/generate
   Headers: { Authorization: Bearer <token> }
   Body: { invoiceId: string }
   Response: { 
     success: boolean,
     data: { 
       ewbNo: number,           // 12-digit E-Way Bill number
       ewayBillDate: string,    // DD/MM/YYYY
       validUpto: string,       // DD/MM/YYYY HH:MM:SS
       alert?: string 
     }
   }
   ```

2. **Get E-Way Bill Details**
   ```
   GET /api/v1/e-waybill/:ewbNo
   Response: { success, data: <complete EWB details> }
   ```

3. **Update Vehicle Number**
   ```
   POST /api/v1/e-waybill/update-vehicle
   Body: {
     ewbNo: number,
     vehicleNo: string,        // Format: MH12AB1234
     fromPlace: string,
     fromState: number,        // State code (1-37)
     reasonCode: '1'|'2'|'3'|'4',  // 1=Breakdown, 2=Transhipment, 3=Others, 4=First Time
     reasonRem?: string,
     transMode: '1'|'2'|'3'|'4'    // 1=Road, 2=Rail, 3=Air, 4=Ship
   }
   ```

4. **Cancel E-Way Bill**
   ```
   POST /api/v1/e-waybill/cancel
   Body: {
     ewbNo: number,
     cancelRsnCode: '1'|'2'|'3'|'4',  // 1=Duplicate, 2=Order Cancelled, 3=Data Entry Mistake, 4=Others
     cancelRmrk: string
   }
   ```

5. **Extend Validity**
   ```
   POST /api/v1/e-waybill/extend-validity
   Body: {
     ewbNo: number,
     vehicleNo: string,
     fromPlace: string,
     fromState: number,
     remainingDistance: number,
     extnRsnCode: '1'|'2'|'3'|'4'|'5',  // Natural Calamity, Law & Order, Transhipment, Accident, Others
     extnRemarks: string,
     consignmentStatus: 'M'|'T',  // M=In Movement, T=In Transit
     transMode: '1'|'2'|'3'|'4'
   }
   ```

6. **Generate Consolidated E-Way Bill**
   ```
   POST /api/v1/e-waybill/consolidated
   Body: {
     tripSheetNo?: string,
     vehicleNo: string,
     fromPlace: string,
     fromState: number,
     transMode: '1'|'2'|'3'|'4',
     ewbNos: number[]  // Array of E-Way Bill numbers
   }
   ```

7. **Reject E-Way Bill** (for recipients)
   ```
   POST /api/v1/e-waybill/reject
   Body: {
     ewbNo: number,
     rejectReasonCode: '1'|'2',  // 1=Duplicate, 2=Wrong Data
     rejectRemark: string
   }
   ```

8. **Get E-Way Bills Generated by Others**
   ```
   GET /api/v1/e-waybill/generated-by-others?date=29/12/2024
   Response: { success, data: [array of EWB objects] }
   ```

### Error Response Format
```json
{
  "success": false,
  "message": "User-friendly error message",
  "error": {
    "errorCode": "2008",
    "message": "Detailed error description"
  }
}
```

## ğŸ”’ Authentication & Authorization

### Setup Axios Instance
```typescript
import axios from 'axios';

const api = axios.create({
  baseURL: process.env.REACT_APP_API_URL || 'http://localhost:3001/api/v1',
  timeout: 30000,
  headers: { 'Content-Type': 'application/json' }
});

// Request interceptor: Add auth token
api.interceptors.request.use(
  (config) => {
    const token = localStorage.getItem('authToken');
    if (token) {
      config.headers.Authorization = `Bearer ${token}`;
    }
    return config;
  },
  (error) => Promise.reject(error)
);

// Response interceptor: Handle errors
api.interceptors.response.use(
  (response) => response,
  (error) => {
    if (error.response?.status === 401) {
      // Token expired or invalid - redirect to login
      localStorage.removeItem('authToken');
      window.location.href = '/login';
    }
    return Promise.reject(error);
  }
);

export default api;
```
